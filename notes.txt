	1c Allow admin management of user accounts

	2a Allow creation of new issues
	2b Describe severity and details of issue
	2c Allow users to edit issues as they develop
3. Bosun tracking of boat issues
	3a Show list of boats
	3b Highlight if issues are repeating
	3c Allow sorting or filtering by severity or boat type affected

	4a Create/delete boats
	4b Edit details of boats
	4c Admins should be able to decide whether or not non-admin users can do this

	5a User creation of new bookings
	5b User shown existing bookings for boats or for dates
